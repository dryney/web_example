Zetaball fuses logic with creativity. If you’re craving a new-age logic game, DOWNLOAD Zetaball today!

Tame gravity and move elements to change the flow of the game ball and hit all the required objects. By successfully hitting all of the objects, you will move onto the next, more challenging round.

ZETABALL FEATURES:
